Pandas Data Analysis – 50 Problems

This repository contains 50 practical data analysis questions solved using only the Pandas library in Python.

The goal of this project is to strengthen core data analysis skills by solving real-world problems involving:

Data Cleaning
Data Filtering
Data Selection
GroupBy Operations
Aggregation
Sorting
Missing Value Handling
Data Transformation
Merging & Joining
Statistical Analysis

Each solution is written in a simple and readable way to improve problem-solving skills and demonstrate proficiency in Pandas.
